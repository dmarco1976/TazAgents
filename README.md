# Tasmania Real Estate Agents Database

Comprehensive database of active real estate sales agents in Tasmania, Australia.

## Current Progress

- **444 agents** collected and verified
- Sources: LJ Hooker, 4one4 Property Co, and others
- Locations: Hobart, Devonport, St Helens, Wynyard

## Data Fields

| Field | Status |
|-------|--------|
| Agent Full Name | Complete |
| Agency Name | Complete |
| Role/Title | Complete |
| Office Location | Complete |
| Mobile Phone | In Progress |
| Email Address | In Progress |
| Agency Website URL | Pending |
| Source URL | Pending |

## Sample Data

See `sample-data.csv` for a preview of the collected data.

## Methodology

- Manual verification from public sources
- No AI-generated or guessed contact details
- Duplicates removed
- Missing information clearly flagged

## Data Sources

- Real estate agency websites
- realestate.com.au agent profiles
- RateMyAgent
- Property Agents Board of Tasmania register
- Business directories

## Deliverable Format

Final delivery available in:
- Microsoft Excel (.xlsx)

- CSV format
